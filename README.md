# Govee.js
 Govee.js is a Javascript Wrapper for the Govee v1.3 API

 # Installation
 ```bash
 npm i govee.js
```

# Usage
Javascript
```javascript
const {Govee} = require("govee.js");
const API = new Govee("API_KEY");
```
Typescript
```typescript
import {Govee} from "govee.js"
const API = new Govee("API_KEY");
```
